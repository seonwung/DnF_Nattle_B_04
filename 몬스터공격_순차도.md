```mermaid
sequenceDiagram
    actor Player as 플레이어
    participant UI as Attack_Monster_UI
    participant Battle as 전투
    participant PlayerClass as 플레이어
    participant Char as 캐릭터 (전사/마법사)

    Player->>UI: 몬스터 공격 요청(플레이어Id)
    UI->>Battle: 몬스터공격(플레이어Id)

    Battle->>PlayerClass: 플레이어체크(플레이어Id)

    alt 플레이어Id == "hero"
        PlayerClass-->>Battle: true (인증 성공)

        alt 직업 == "전사"
            Battle->>Char: 스킬발동_검휘두르기()
            Char-->>Battle: 데미지 반환 (공격력 * 1.5)
        else 직업 == "마법사"
            Battle->>Char: 스킬발동_파이어볼()
            Char-->>Battle: 데미지 반환 (공격력 * 2.0)
        end

        alt 데미지 >= 200
            Battle->>Battle: 등급 = "S급 공격"
        else 데미지 >= 100
            Battle->>Battle: 등급 = "A급 공격"
        else 데미지 < 100
            Battle->>Battle: 등급 = "B급 공격"
        end

        Battle-->>UI: 공격 결과 반환(데미지, 등급)
        UI-->>Player: 공격 성공 및 등급 출력

    else 플레이어Id != "hero"
        PlayerClass-->>Battle: false (인증 실패)
        Battle-->>UI: 오류 메시지(권한 없음)
        UI-->>Player: 공격 실패 알림
    end
````
